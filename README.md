# Оно тебе надо
Аукцион вещей, в которые никто не верил. Учебный первый проект

## Описание проекта
Одностраничный лендинг аукциона. Сайт представляет информацию о необычных лотах, об аукционе

## Технологии
- HTML5
- CSS3 (Flexbox, Grid)
- Адаптивная вёрстка

## Структура проекта

проект/
├── index.html
├── styles/
│ ├── fonts.css
│ ├── global.css
│ └── style.css
├── images/
└── fonts/

## Секции сайта
- Шапка с навигацией и контактами
- Обложка с заголовком и кнопкой
- Карточки лотов
- Секция "Об аукционе"
- Подвал с навигацией и соцсетями
